G-Code is the fundamental language of CNC Programming and the features of this application are quite simple. The program is built to detect lines while coding the detected lines at the same time. Similar to CAM software, this software in particular allows user to draw out using line and dimension input to create desired shape. The software will detect below image made up of straight lines, turn in to G-code which is a language to tell a Plasma and Oxy CNC machine what to do or how to do it. The G-code commands itself instruct the CNC machine where to move, how fast to move and what path to follow. The program provides basic shape like, square, triangle and circles. Depending on types of material, for example the thickness, type of metal like the ferrous Metals (mild steel, medium carbon steel), and Non-ferrous metals (aluminium, copper, brass.
